# Contributing

> **Governance:** DGAF / Agent Amethyst — All changes to this repository are subject to Sentinel CI/CD integrity enforcement. Contributions must pass governance checks before merge. See [DGAF-Framework](https://github.com/Flickerflash/DGAF-Framework) for spine documentation.

## Scope
This repository contains validated AI governance frameworks: Paul-Elder Critical Thinking, NIST AI RMF, ISO 42001, and IIA AI Auditing — with implementation roadmaps and success metrics.

## Development
- Keep changes small and reviewable.
- Framework updates must cite the authoritative source (NIST, ISO, IIA, or OWASP).
- Implementation roadmap changes must maintain traceability to the source standard.

## Pull Requests
- Identify which framework component is affected (NIST, ISO 42001, Paul-Elder, IIA).
- Note whether the change affects compliance mappings, success metrics, or implementation guidance.
