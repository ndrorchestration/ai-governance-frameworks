# NT-03: KB Answer With Quality Check

## Purpose

Combines Retrieval-Augmented Generation (NT-02) with an integrated quality
evaluation pass (NT-01), producing a complete audit-ready evidence chain.
This template spans both P-05 (Tri-Phase CI Gate) and P-11 (11Q Attestation
Scoring) in sequential flow, satisfying NIST AI RMF MEASURE 2.9 and
ISO 42001 §9.1.

## Sequential Flow (P-05 → P-11)

```
[Retrieve]  ← P-05 Phase 1
    │
[Generate]  ← P-05 Phase 2
    │
[Evaluate]  ← P-05 Phase 3 → triggers P-11
    │
[11Q Score] ← P-11 (11Q Attestation Scoring)
    │
[Audit Record] — stored per run as evidence
```

---

## IIA AI Auditing Relevance

This template produces the evidence chain required for AI audit readiness:
- Source retrieval log (what was retrieved and from where)
- Generation record (what was produced and from which context)
- Quality score (11Q dimensions, scored and stored)

This makes each run independently auditable without requiring access to the
underlying LLM.
